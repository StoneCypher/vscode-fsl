# Code of Conduct

## Our standard

Be respectful, constructive, and professional. We are here to build good software together.

## Expected behavior

- Give and receive feedback gracefully
- Assume good intent
- Stay focused on the technical problem
- Credit others' contributions

## Unacceptable behavior

- Personal attacks, insults, or harassment
- Discriminatory language or conduct
- Publishing others' private information
- Sustained disruption of discussions

## Enforcement

Project maintainers will remove, edit, or reject contributions that violate this code of conduct. Repeated or severe violations may result in a ban from the project.

## Reporting

Report issues to the project maintainers. All reports will be reviewed and handled with discretion.
